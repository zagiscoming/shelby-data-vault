# Contributing to Shelby Data Vault

First off, thanks for taking the time to contribute!

## How Can I Contribute?

### Reporting Bugs

*   Ensure the bug was not already reported by searching on GitHub under [Issues](https://github.com/zagiscoming/shelby-data-vault/issues).
*   If you're unable to find an open issue addressing the problem, open a new one.

### Suggesting Enhancements

*   Open a new issue and describe the enhancement you'd like to see.

### Pull Requests

*   Fill in the required template.
*   Do not include issue numbers in the PR title.
*   Include screenshots and animated GIFs in your pull request whenever possible.

## Styleguide

*   We use `prettier` for code formatting.
*   Follow the existing code style in the project.
