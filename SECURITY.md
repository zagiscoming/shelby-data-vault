# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of Shelby Data Vault seriously. If you believe you have found a security vulnerability, please report it by opening a GitHub issue or contacting the maintainers directly.

Please include:
- A description of the issue.
- Steps to reproduce.
- Potential impact.

We will acknowledge your report and work on a fix as quickly as possible.
